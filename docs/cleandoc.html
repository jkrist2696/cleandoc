Run clean_pyfile function on all .py files in searchpath.

Parameters:

  • searchpath (str) -- Directory to search in all nested folders for .py files

  • write (bool) -- keyword argument passed to clean_pyfile function

  • skip (Union[bool, List[str]]) -- List of .py files to skip cleaning. Or True to find list of clean pyfiles within function.
